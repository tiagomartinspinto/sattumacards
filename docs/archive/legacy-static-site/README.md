# Legacy static site archive

These files are from the older pre-Express static version of Sattuma.

They are kept only as historical reference material and are not part of the live application runtime. The current app entrypoint is served from `public/index.html` through the Node/Express server.

The archived files may not run cleanly as a standalone site without adjusting older relative paths, so they should be treated as reference snapshots rather than supported entrypoints.
