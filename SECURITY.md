# Security Policy

## Reporting a vulnerability

If you discover a security issue in Sattuma, please contact the maintainers directly instead of opening a public issue with exploit details.

Current contact addresses:

- tomi.slottedufva@aalto.fi
- mikael.brygger@uniarts.fi
- tomi.humalisto@uniarts.fi
- tiago.martins@aalto.fi

Please include:

- a clear description of the issue
- reproduction steps
- impact assessment if known
- any suggested mitigation
